$ git diff > hoge.patch

hoge.patchを開くと下記のような制御コードのまじった結果になりpatchとして使用できない。

^[[0m^[[01;32march^[[0m      ^[[01;32mcut^[[0m                   ^[[01;32mdoexec^[[0m      ^[[01;32mgettext^[[0m   ^[[01;36mksh^[[0m

$ b git diff > hoge.patch
